# aliases

Aliases


For `.sh` files, you'll need to run `chmod 755 main.sh` or whatever.


In `.bashrc` add:

```sh
export PATH="$HOME/GitHub/aliases:$PATH"
```

Other stuff to put in that file:

```sh
export PATH="$HOME/GitHub/aliases:$PATH"
alias dir='ls'
alias explorer='nautilus'
alias cd..='cd ..'
alias cls='clear'
alias csl='clear'
alias home='cd ~/GitHub
clear'
```
